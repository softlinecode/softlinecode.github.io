YUI Compressor 를 이용해서 JS 를 압축하고 다시 하나의 파일로 머즈하는것을
적용해 보았습니다.

우선 기존에는 자체 tool로 이용해서 머즈를 하였으나 머즈를 주로 하지 않는 개발자의
경우 머즈를 하려면 여러가지를 설치하고 설정해야하는 번거로움이 있었습니다.
하지만 이렇게 pom.xml 파일에 기술하고 머즈 빌드를 하게 되면 필요한 머즈된 js 파일을
얻게 됩니다.

우선 해당 방법은 maven 을 사용할줄 안다는 가정하에 정리를 합니다.

   <artifactId>yuicompressor-maven-plugin</artifactId>
   <version>1.3.0</version>

      <id>compress</id>
              <phase>compress</phase>

         <goal>compress</goal>
       </goals>

       <disableOptimizations>false</disableOptimizations>
