Maven , Eclipse  , Eclipse plugin 설치 완료후


workspace 폴더로 이동후
D:\dev\template-corexp\workspace>
mvn archetype:create -DgroupId=com.cors.co -DartifactId=corexp -DarchetypeArtifactId=maven-archetype-webapp

해당 명열어 실행

mvn = 내컴퓨터에 설정된 maven 명령어 (현재 환결설정 path에 maven 실행 bin 파일을 연결해놓았음)
archetype:create = maven에서 archetype이라는 기본 구조로 프로젝트를 생성할수 있는 것을 제공한다. create는 현재 maven 구조의 프로젝트를 생성하라는 명령어
-DgroupId=com.cors.co = groupId를 지정한다.
-DartifactId=corexp = artifactId를 지정한다.
DarchetypeArtifactId=maven-archetype-webapp = archetypeArtifactId는 생성하려는 프로젝트의 종류를 설정한다. 현재 maven에서는 여러가지의 종류별 archetype이 존재한다.

생성후 eclipse에서 프로젝트를 정상적으로 작업하기 위해서는 해당 workspace 로 들어가서
mvn eclipse:eclipse 명령어를 실행한다. 해당 프로젝트를  eclipse wtp에 인식하도록 설정을 해준다.

D:\dev\template-corexp\workspace\corexp>mvn eclipse:eclipse
