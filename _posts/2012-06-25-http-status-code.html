간단하게 정리된 사이트 주소
http://status-code.com/

Hypertext Transfer Protocol -- HTTP/1.1  주소
http://www.w3.org/Protocols/rfc2616/rfc2616.html

주로 검색하는 CODE

403 Forbidden
404 Not found
500 Internal Server Error

보통 세가지 코드가 대표적이다.

403는 접근하려는 곳에 대해서 접근 권한이 없는 것이다. 접근 권한에 대해서 조사해 봐야 한다.
404는 해당 Request를 대해서 경로를 못 찾는 것이다. 쉽게 말하면 잘못된 url 이 겠죠.
500는 서버에서 에러가 발생하죠. 서버의 에러 내용까지 볼 수 있을 것이다.

대표적인 3가지 경우는 보았지만 실제로 Spec 문서를 보면 다양한 코드와 방대한 내용들이 있다.

영어의 압박을 다시 한번 느끼게 한다.
