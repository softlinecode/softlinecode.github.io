  1. 자료형
프로그램을 처음 접하게 되면 그 언어의 자료형부터 파악을 해야합니다. 대부분의 언어들이 갖고 있는 자료형은 비슷한 점이 많습니다. 아마도 하나 이상의 언어를 습득하신 분들이라면 자료형 부분에서는 Python의 자료형은 이렇구나 라고 생각하시면 됩니다. Python 이 쉬운 script 언어로써 많이 사랑 받는 이유를 자료형을 보면 알수 있을 정도로 간단하고 사용하기 편리하도록 되어져 있습니다.
   
     1-1) 숫자형(Number)      
    숫자형이란 숫자 형태로 이루어진 자료형입니다. 일반적으로 숫자라고 표현하는 것 이라고 생각하면 됩니다.  


항목          사용 예
----------------------------------
정수          12345, -12345, 0
실수          123.45, -1234.5
복소수        1 + 2j, -3j
8진수         034
16진수        0x2A, 0xFF
  • 정수형(Integer) - 정수인 숫자형입니다. 양의 정수와 음의 정수 와 0 으로 이루어져 있습니다. ( 2,-2, 0)
  • 소숫점 포함된 숫자(float) - 소수점이 포함된 숫자라고 보시면 됩니다. (123.45)
  • 8진수(Octal) - 8진수로 이뤄진 숫자이며 0 으로 시작됩니다.(0177)
  • 16진수(Hexadecimal) - 16진수로 이뤄진 숫자이며 0x로 시작됩니다.

숫자 연산

덧셈(+), 빼기(-), 곱하기(*), 나누기(/)
>>> a = 3
>>> b = 5
>>> a + b
8
>>> a - b
-2
>>> a * b
15
>>> a / b
0

나머지(%) - 나누기를 한 나머지를 보여줍니다.
>>> a % b
3
