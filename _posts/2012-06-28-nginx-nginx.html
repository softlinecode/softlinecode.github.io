그 중 가장 많이 사용되고 대부분 web server 하면 떠올리는 것은 apache web server 입니다. 하지만 최근에 nginx 라는 웹서버가 많은 관심을 받고 있습니다.

Nginx 란
1.오픈 소스 웹 서버
2.Reverse Proxy 서버와 IMAP/POP3 프록시 서버 기능 제공
3.비동기 이벤트 드리븐 방식으로 요청을 처리
4.쓰레드(프로세스)지향적으로 이벤트를 처리하는 Apache 에 비해 효율적인 성능을 가짐.

러시아의 Igor Sysoev가 2002년에 처음 선보인 고성능의 웹 서버(2004년 첫 public release).
현재 최신 Stable 버전 v1.0.14.
