--- layout: post title: linux max file open limit date: '2012-11-21T01:35:00.001-08:00' author: Jung Gi Lim tags: - linux modified_time: '2012-11-21T01:35:07.186-08:00' blogger_id: tag:blogger.com,1999:blog-2993388283610032812.post-251465300645254832 blogger_orig_url: http://softline21c.blogspot.com/2012/11/linux-max-file-open-limit.html --- 간혹 linux 사용시 해당 계정이 open 할수 있는 파일의 갯수 제한이 있어서
해당 limit 제한을 늘려야 하는 경우가 있다.
limits.conf 파일에 해당 계정의 file open limit를 수정하면 된다.

vi /etc/security/limits.conf

root             soft    nofile          16000
root             hard    nofile          16000
