Java + eclipse는 이미 설치가 되어져 있다고 생각하고 Python 부분만 정리합니다.

우선 Python을 설치해야 합니다.

http://www.python.org/download/

해당 URL로 가시면 버젼별(OS 환경별)로 설치 할수가 있습니다.
다운 받으시고 default로 그냥 설치하시면 됩니다.
저는 window7(64bit) 인데 그냥 next .. 하니 설치가 끝났네요.

이제 eclipse plugin을 설치해야 합니다.
Pydev 를 사용하도록 하겠습니다.
http://pydev.org/

eclipse plugin 설치 url
http://pydev.org/updates
설치 가이드 url
http://pydev.org/manual_101_install.html

Help -> Install New Software  에서 plugin url을 통해서 설치하시면 됩니다.

plugin 이 설치가 되면
Window -> Preferences -> PyDev -> Interpreter - Python
에 가서 New 버튼을 이용해서 설치된 Python 실행파일(python.ext) 선택후 라이브러리도
같이 선택하도록 합니다.
