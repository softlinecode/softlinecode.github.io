node.js를 사용하다보면 npm 이라는 것을 이용해서 설치하는게 나온다. 처음에는 설치 안하고 했으나
결국 npm은 필수 인것 같다. node.js를 사용하실 분은 필히 처음부터 npm을 설치하시고 작업하시길 그러지 않으면 나중에 저처럼 삽질 열심히 하고 후회하게 됩니다.

간단하게 
curl http://npmjs.org/install.sh | sh
root 로 실행하지 않으면 중간에 파일 access 권한 문제로 잘 안될수 있으니 root 실행해야한다.
실행시 기존버젼 제거에 대해서 물어보면 yes를 입력하면 된다.

다른 여러 방법및 자세한 설명은 밑에 site에서 확인할수 있다.
