현재 최신 버젼(Helios) 을 다운 받는다.
URL: http://eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/heliosr

대부분의 최근에 환경은 UTF-8로 개발을 할것이다.
하지만 eclipse 개발의 default는 MS949 로 되어져 있다. 처음에 변경하지 않으면 잘못된 수정값으로
나중에 고생할수 있다.

eclipse -> Window -> Preferences -> General  -> Worksspace 
에서 Text file encoding 을 UTF-8로 수정
(찾기 input 에 encod 정도만 치면 encoding 관련된 설정 값을 볼수 있다)



ecilpse -> Window -> Preferences -> General -> Editors -> File Associations
부분은 파일 연관성에 관한 부분입니다. 
보통 css 파일이 클경우 해당 파일을 열게 되면 ecilpse가 느려지게 되거나 다운될때가 있는데요
그걸 방지하기 위해서 *.css 파일의 경우 Text Editor로 열리도록 수정
밑에 Class 파일의 경우 Jad 를 이용해서 DeCompile 합니다.




Jad , JadClipse를 이용해서 Decompile 한다. 관련해서 http://lyb1495.tistory.com/56 참고
*.class 파일에 대해서 Jad를 이용한다.



Jad 실행 파일 위치 (Path to decompiler)
eclipse -> preferences -> java -> jadClipse



eclipse -> window -> preferences -> general -> editor -> text editors -> Spelling
에서 자신에 맞게 설정한다.



eclipse -> preferences -> java -> installed JRE's 
에서 eclipse가 실행될때 필요한 JRE를 지정한다.



eclipse -> preferences -> general -> editors -> Text Editors
에서 라인수를 보여주거나 tab width를 정할수 있다.
