파이썬(Python) 놀아보기

예전부터 파이썬(Python) 을 가지고 놀아 보려고 합니다.
명성만 들어봤고 예전에 간단하게 책을 조금 본적이 있는 수준입니다.


1. 파이썬 설치
  크롬앱에 Python Shell 이 설치가 됩니다.  간단하게 우선 설치해서 테스트 해보고 싶은 초보분들에게는 아주 유용 합니다.
  
크롬에서 https://chrome.google.com/webstore/detail/gdiimmpmdoofmahingpgabiikimjgcia
치면 밑에 화면이 나오고 ADDED TO CHROME 버튼을 클릭하면 다운되고 설치도 합니다.
















설치후 크롬 부라우져 오른쪽에 파이썬 앱 아이콘이 있습니다
클릭하시면 레이어 형태로 보이고요 저는 팝업으로 변경해서 사용하였습니다.




















이렇게 간단하게 Python Shell 테스트 환경을 크롬의 도움으로 쉽게 시작할수 있습니다.
다음에는 직접 window , linux 에 설치해서 사용하는 방법도 정리해보록 하겠습니다.
