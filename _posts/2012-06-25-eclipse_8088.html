자주 사용하는 단축키의 경우 잘 동작하지 않는 경우가 있습니다.  새로운 plugin을 설치 한 후에 주로 이러한 현상들이 일어나는데요. 특별하게 원인을 찾지 못하는 경우가 많습니다. 사용자 정의로 단축키를 등록해도 안되는 경우도 있습니다. eclipse 관련 해서 공부하다가 알게 된 것입니다.

eclipse -> window -> Customize Perspective -> Command Groups Availability 탭에서 해당
plugin 이 선택되어져 있는지 확인해 보세요. 선택되어져 있지 않다면 선택후 확인(OK)를 하시면 해당
plugin 에 단축키가 동작할 겁니다.

저는 SVN 의 단축키가 되지 않아서 한동안 고생을 했는데요. 해당 방법으로 해결했습니다.
원인을 못찾고 그냥 사용하시는 분들 생각보다 많더라고요.(새로 설치하시는 분들도 있고요)
