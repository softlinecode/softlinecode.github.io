    문자로 이루어진 문장을 말한다. 프로그램을 하면서 가장 많이 사용되는 자료형이고 사용하는데 어려운 부분은 없다.

  • 문자열 생성 규칙
      "" , ''  ,""" """ , ''' ''' 으로 둘러싸져 있으면 문자열이 됩니다.
      "Hello World"
      'Hello World'
      '''Hello World'''
      """Hello World"""


  • 문자열 내에 (") , (') 포함

     역슬래쉬(\)를 이용해서 \' , \"  이렇게 되면 해당 문자열안으로 인식됩니다.

  • 문자열 연산

     파이썬에서는 문자를 더하기(+) 도 되고 특이하게 곱하기(*)도 된다.

     1) 문자열 합치기 - 문자를 + 표시로 더하게 되면 문자가 합쳐지게 됩니다.
      >>> a = "Hello"
      >>> b = "World"
      >>> a + b
      HelloWorld

     1) 문자열 곱하기 - 문자를 * 표시로 곱하기를 하면 그 수 만큼 더하기가 됩니다.
      >>> a = "Hello"
      >>> print a * 2
      HelloHello
  • 문자열 인덱싱(indexing)

     파이썬에서는 문자열에 인덱싱(indexing) 기능이 있습니다.
     문자열에 배열구문처럼 사용하여 해당 index에 값을 가져올수 있습니다.
     파이썬의 index는 0 부터 시작합니다.
     >>> a = "Hello"
     >>> print a[0]
     H
 
     문자열 인덱싱에서 실수로 값을 변경 가능하다고 생각할 수 있습니다. 하지만 문자열 인덱싱으로는 해당 인덱싱 값만 가져올수 있지 값을 변경할 수 없습니다.
    >>>a = "Hello"
    >>>a[0] =  "B"
    >>>print(a)
   Hello   => B로 변경되지 않습니다.


    문자열 인덱싱에서는 -1 이 가능합니다. 미아너스는 결국 뒤로 인덱싱 한다는 뜻입니다.
    >>>a = "Hello"
    >>>print(a[-1])   
    o => 0 인덱싱에서 뒤로 가니깐 가장 끝에 있는 문자를 가져오게 됩니다.
    

  • 문자열 슬라이싱
     파이썬에서는 문자열 슬라이싱 기법이라는 방법을 제공합니다. 저도 이런 기법은 처음 접해 보는 내용인데 참으로 마음에 드는 기법입니다. 아마도 이 기법으로 코드량이 많이 줄어들 것으로 기대가 됩니다.
  
  >>>a ="Hello"
  >>>print(a[0:3])
   Hel
  >>>print(a[1:2])
  e
  >>>print(a[0:2])
  He
   
  위에 예제를 먼저 보여드렸습니다. 대충 이해가 되시죠? 네 아마도 이것을 보시는 분들이 아 이런거구나 그 생각도 맞습니다. 문자에서 [시작번호:끝번호] 로 인식을 해서 해당 문자열을 가져오게 됩니다. 물론 - 인덱싱도 가능합니다.


  >>>a="20100101" 
  >>>print(a[:4])
  >>>print(a[4:6])
  >>>print(a[6:])


  날짜의 경우 년,월,일로 substring 하는 경우보다는 깔끔하네요. 이런 용도로 많이 사용될 것 같습니다.



  • 문자열 함수
     >>> a = "Hello"
     >>> a.upper()
     HELLO -> 대문자로 변환
  
    >>> a.count('l')
     2 -> Hello 문자중에서 ㅣ 이 몇개 존재하는지 count


     >>> a.find('l')
     2 -> Hello 문자중에서 ㅣ의 첫번째 index를 반환한다. 존재하지 않으면 -1 발생
   
     >>> a.index('l')
     2 -> Hello 문자중에서 ㅣ의 첫번째 index를 반환한다. 존재하지 않으면 에러를 발생한다.
    
    함수 정리

a.upper() 문자열 a를 모두 대문자로 바꾸어 준다.
a.count(x) 문자열 a중 x와 일치하는 것의 갯수를 반환한다.
a.find(x) 문자열 a중 문자 x가 처음으로 나온 위치를 반환한다. 없으면 -1을 반환한다.
a.index(x) 문자열 a중 문자 x가 처음으로 나온 위치를 반환한다. 없으면 에러를 발생시킨다.
a.join(s) s라는 문자열의 각각의 요소 문자사이에 문자열 a를 삽입한다.
a.lower() 문자열 a를 모두 소문자로 바꾸어 준다.
a.lstrip() 문자열 a의 왼쪽 공백을 모두 지운다.
a.rstrip() 문자열 a의 오른쪽 공백을 모두 지운다.
a.strip() 문자열 a의 양쪽 공백을 모두 지운다.
a.replace(s, r) 문자열 a의 s라는 문자열을 r이라는 문자열로 치환한다.
a.split([s]) 문자열 a를 공백으로 나누어 리스트값을 돌려준다.
a.swapcase() 문자열 a의 대문자는 소문자로, 소문자는 대문자로 각각 바꾸어 준다. 
  
   
  

  • 문자열 포매팅
     print 를 이용해서 문자열을 출력할때 변수로 대입을 해서 출력가능합니다.
    
   >>> print "Hi %s" % "yang"
   HI yang


   이렇게 %s 로 대입되는 곳을 지정후 문자열 종료후 % 이후에 해당 값을 지정하면
  이렇게 문자열을 완성해서 출력하게 됩니다.
  숫자의 경우 %d 를 사용 가능합니다. 하지만 보통 모든 자료형이 문자열로 대신 하니깐
 주로 %s를 사용하게 될 것입니다.


  >>> print "Hi %s , Hello %s" %("yang" , "kim")
  Hi yang , Hello kim

  이렇게 복수의 대입값도 사용가능합니다.
